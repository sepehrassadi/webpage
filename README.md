# sepehr.assadi.info

Source for my academic website, served via GitHub Pages at https://sepehr.assadi.info.

Static hand-written HTML — no build step. Editing the `.html` files and pushing to the
default branch publishes the site.

## Layout
- `index.html` — the main page (bio, teaching, mentoring, activities, publications, talks, links).
- `courses/` — one folder per course, each with its own `index.html`.
- `papers/`, `presentations/` — publication and slide PDFs.
- `files/` — CV (`sepehr-assadi.pdf`) and third-person bio (`bio.txt`).
- `images/` — headshot and figures.
- `sosa26/`, `space-bounded-computation-reading-group/` — standalone event / reading-group pages.
- `old/` — archived previous version of the site.
- `CNAME` — custom domain configuration for GitHub Pages.
